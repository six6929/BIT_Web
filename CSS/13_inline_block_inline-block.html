CSS display : block, inline, inline-block

div 태그는 block 속성

block속성은 브라우저 좌측부터 우측 끝까지 영역을 차지

block속성은 크기를 가짐 - width, height 속성으로 크기 지정

div는 block 영역

span 태그는 inline 속성

inline속성은 한 라인내에서 영역을 차지

inline속성은 내용크기로 정해짐 - width, height 지정해도 크기가 변하지 않음

span은 inline 속성그래서 한줄에 표시된다. 크기를 지정해도 크기가 달라지지 않는다.

inline-block 속성

inline-block속성은 한 라인내에서 영역을 차지(inline)

inline-block속성은 크기를 지정하면 적용됨(block)

inline-block은 전체영역을 차지하지 않고
가로, 세로 크기
한줄에 표시되지만, 크기를 가질수 있다.
